# Лента.PM — Система управления проектами

Внутренняя система управления проектами для команды реконструкций и констракшн.
Разработана специально под процессы и форматы данных компании ЛЕНТА.

---

## Что умеет система

| Модуль | Возможности |
|--------|-------------|
| **Проекты** | Импорт из Excel, статусы, этапы, дедлайны, экспорт |
| **Задачи** | Постановка, приоритеты, уведомления, статусы |
| **ВПК-отчёты** | Чеклисты ВПК-1/2, фото, экспорт в Excel |
| **Аналитика** | Статистика открытий, просрочки, дашборд |
| **Чат** | Личные и общий чат, фото, онлайн-статусы |
| **ИИ-ассистент** | Claude, Groq, DeepSeek с контекстом проектов |
| **Дедлайны** | Единая лента проектов и задач по срокам |
| **КСО** | Объекты, готовность, графики работ |
| **Синхронизация** | Автоматический импорт из Excel по расписанию |

---

## Технологии

- **Backend:** Python 3.11+ / FastAPI
- **База данных:** PostgreSQL (или SQLite для разработки)
- **Frontend:** Jinja2 + Bootstrap 5 + Vanilla JS
- **Деплой:** Railway.app / любой Linux-сервер
- **Безопасность:** PBKDF2, WebAuthn, CSRF, rate limiting

## Быстрый старт (локально)

```bash
# 1. Клонировать репозиторий
git clone <repo-url>
cd lenta-web

# 2. Создать виртуальное окружение
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Настроить переменные
cp .env.example .env
# Открыть .env и заполнить SECRET_KEY и DATABASE_URL

# 5. Запустить
uvicorn main:app --reload

# 6. Открыть в браузере
# http://localhost:8000
```

## Как добавить пользователя

1. Войти как администратор
2. Открыть `/admin/users`
3. Нажать "Добавить" → ввести телефон в формате +7XXXXXXXXXX
4. Пользователь при первом входе создаёт пароль самостоятельно

## Переменные окружения

Шаблон: **[.env.example](.env.example)**

| Переменная | Обязательная | Описание |
|-----------|:---:|---------|
| `SECRET_KEY` | ✅ | Ключ подписи сессий |
| `DATABASE_URL` | ✅ | Строка подключения PostgreSQL |
| `ADMIN_PHONE` | ✅ | Телефон первого администратора |
| `APP_DOMAIN` | ✅ | Домен (для Face ID) |
| `GROQ_API_KEY` | — | ИИ-ассистент (бесплатно) |
| `SMTP_HOST` | — | Email-уведомления |
