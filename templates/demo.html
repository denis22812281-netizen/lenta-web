Возможности

Всё что нужно команде

Полный цикл управления объектами — от импорта Excel до аналитики открытий.

Импорт из Excel
Загружаете файл — данные обновляются автоматически. Умный парсер не ломается при изменении структуры файла.
Дедлайны
Единая таблица дедлайнов по проектам и задачам. Цветовая индикация: просрочено, горит, норма.
ВПК-отчёты
Чек-листы ВПК 1 и ВПК 2 с фото к каждому критерию. Автоматическая отправка email руководителям.
Статистика открытий
Раньше срока / вовремя / с опозданием — по каждому менеджеру и объекту. Экспорт в Excel.
Внутренний чат
Общий и личный чат с отправкой фото. Онлайн-индикаторы, счётчики непрочитанных сообщений.
ИИ-ассистент
Встроенный чат с ИИ (Claude / Llama / DeepSeek), который знает актуальные данные системы.
Задачи
Постановка задач с дедлайном и исполнителем. Email при назначении и завершении. Фото-отчёт.
Менеджеры
Страница команды с фото, статистикой по объектам, онлайн-статусом и срочными проектами.
Безопасность
Вход по телефону + пароль. Белый список пользователей. Face ID / Touch ID. CSRF, rate limiting.
