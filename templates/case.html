Контекст

Строительный департамент — хаос в Excel

Реконструкции и стройки по всей России. Каждый менеджер ведёт свои объекты сам. Руководитель не видит картину целиком. Решения принимаются на основе устаревших данных.

Как было
  • Статус объектов — звонок каждому менеджеру
  • Дедлайны в Excel, обновляются вручную раз в неделю
  • Задачи ставятся в WhatsApp, теряются
  • Уведомления подрядчикам — вручную по email
  • Отчёт для руководителя — 2–3 часа сбора данных
  • Нет единой картины по портфелю проектов
Стало
  • Статус всех объектов — в реальном времени на дашборде
  • Дедлайны с цветовой индикацией, импорт из Excel
  • Трекер задач с приоритетами и исполнителями
  • Автоуведомления подрядчикам с подтверждением
  • Отчёт для руководителя — кнопка «Экспорт»
  • AI-ассистент отвечает на любой вопрос по портфелю
Что построено

Полноценная PM-система с нуля

8 функциональных модулей, AI-аналитика, права доступа, мобильный интерфейс.

Дашборд
Сводка в реальном времени: активные, просроченные, срочные объекты. Онлайн-присутствие команды.
Реестр проектов
Реконструкции и Констракшн. Фильтрация, поиск, массовые действия. Карточка объекта с историей.
Дедлайны
Gantt-таблица этапов по всем объектам. Цветовая индикация. Импорт из Excel одним файлом.
Задачи
Внутренний трекер команды. Приоритеты, исполнители, сроки. Замена WhatsApp-поручениям.
SMR-уведомления
Автоматические письма подрядчикам с подтверждением через ссылку. База контактов.
AI-ассистент
Видит все проекты в реальном времени. Отвечает на вопросы по портфелю. Groq / Claude / DeepSeek.
Статистика
Аналитические графики: распределение по статусам, нагрузка менеджеров, динамика завершений.
Безопасность
Авторизация по телефону, CSRF-защита, rate limiting, аудит действий, разграничение прав.
