1. Назначение системы

Система Лента.PM — внутренний инструмент строительного департамента сети ЛЕНТА. Предназначена для полного контроля жизненного цикла объектов реконструкции и нового строительства.

  • Отслеживание статусов объектов в реальном времени
  • Контроль дедлайнов по этапам проектов
  • Управление задачами внутри команды
  • Отправка SMR-уведомлений подрядчикам и получение подтверждений
  • Формирование отчётов и экспорт в Excel одним кликом
  • AI-ассистент с доступом ко всем данным системы
Команда: руководитель Гаврин Игорь + 8 менеджеров проектов. Доступ предоставляется администратором по номеру телефона.

2. Вход в систему

Первый вход

  1. Откройте браузер и перейдите по ссылке системы (можно отсканировать QR-код)
  2. Введите номер телефона в формате +7XXXXXXXXXX
  3. Если номер есть в системе — введите пароль (при первом входе создайте новый)
  4. Нажмите «Войти» — откроется главный дашборд
Совет: Сохраните пароль в браузере — кнопка «Сохранить» появится при вводе автоматически.

Если забыли пароль

Обратитесь к администратору — Месмер Денис. Он сбросит пароль, при следующем входе вы создадите новый.

3. Дашборд — главная страница

После входа открывается дашборд — сводка по всему портфелю проектов в реальном времени.

  • Карточки статистики — активные, завершённые, просроченные объекты
  • Срочные объекты — дедлайн менее 7 дней
  • Просроченные — объекты с истёкшим сроком и количество дней просрочки
  • Открытые задачи — задачи команды, требующие внимания
  • Онлайн — кто из коллег сейчас работает в системе
Красный — просрочено Жёлтый — менее 7 дней Зелёный — в норме

4. Проекты — Реконструкция

Раздел содержит все объекты типа «Реконструкция» с фильтрацией, поиском и массовыми операциями.

Список объектов

  • Фильтрация по статусу, менеджеру, городу
  • Поиск по номеру ТК или названию объекта
  • Массовые действия — выбрать несколько объектов и изменить статус или менеджера

Статусы проекта

Планирование Активный Приостановлен Завершён

Создание нового объекта

  1. Нажмите «+ Добавить» в правом верхнем углу списка
  2. Заполните обязательные поля: название, номер ТК, город, тип, статус
  3. Назначьте ответственного менеджера и укажите плановые даты
  4. Нажмите «Сохранить»

Карточка объекта

Нажмите на объект для просмотра деталей: основные данные, этапы проекта, файлы, история изменений.

5. Проекты — Констракшн

Раздел для объектов нового строительства. Интерфейс и логика работы идентичны разделу «Реконструкция».

Разница: в таблицах дедлайнов и отчётах Реконструкции и Констракшн отображаются на отдельных вкладках.

6. Задачи

Внутренний трекер команды для оперативного взаимодействия и постановки поручений.

Создание задачи

  1. Перейдите в раздел «Задачи» в боковом меню
  2. Нажмите «+ Новая задача»
  3. Укажите название, описание, приоритет, исполнителя и срок выполнения
  4. Нажмите «Сохранить»

Приоритеты

Низкий Средний Высокий
Совет: При завершении задачи добавьте комментарий — это помогает вести историю работы по объекту.

7. Дедлайны

Сводная таблица по этапам всех активных проектов — видно что просрочено, что завершено, что в работе.

Как читать таблицу

  • Строки — объекты (ТК)
  • Столбцы — этапы проекта с датами начала и окончания
  • Цвет ячейки показывает состояние этапа (просрочено / скоро / выполнено)

Импорт из Excel

  1. Подготовьте файл в установленном формате
  2. Перейдите «Дедлайны» → «Импортировать»
  3. Выберите файл (макс. 10 МБ) и нажмите «Загрузить»
  4. Система обновит даты автоматически по всем объектам

8. SMR — строительно-монтажные работы

Модуль для управления графиками подрядчиков и автоматической отправки уведомлений о выполненных работах.

Отправка уведомления

  1. Откройте нужный SMR-объект из списка
  2. Отметьте выполненные этапы работ
  3. Нажмите «Отправить уведомление» — подрядчик получит письмо
  4. Подрядчик переходит по ссылке и подтверждает выполнение
Подтверждение фиксируется в системе с датой и временем — для отчётности и контроля.

База контактов

Раздел «Контакты» → список подрядчиков с email-адресами. Можно добавлять и удалять контакты.

9. ВПК — внутренний контроль

Мониторинг отчётов по объектам для внутреннего контроля качества.

  • Реестр отчётов с датами и статусами по объектам
  • Счётчик непрочитанных материалов в боковом меню
  • Экспорт реестра в Excel
Индикатор ВПК: цифра рядом с пунктом меню показывает количество новых непрочитанных отчётов.

10. КСО — отчётность

Реестр КСО-документации по объектам — хранение и контроль отчётных документов.

  • Фильтрация по объекту, дате, статусу проверки
  • Прикрепление файлов к записям
  • Отметки о прохождении проверки

11. Менеджеры

Справочник команды — контактные данные и текущая нагрузка по каждому менеджеру.

  • ФИО, телефон, email каждого сотрудника
  • Количество активных объектов у менеджера
  • Отметка руководителя (Гаврин Игорь)

Менеджеры из этого справочника назначаются на объекты в карточках проектов.

12. AI-ассистент

Встроенный чат-бот, который видит все данные системы в реальном времени и помогает анализировать портфель проектов.

Что умеет

  • «Сколько у нас просроченных объектов?»
  • «Какие ТК закрываются в этом месяце?»
  • «У кого из менеджеров больше всего активных проектов?»
  • «Составь сводку по статусам портфеля»
  • Проверить загруженный Excel-файл на ошибки в формулах

Провайдеры

Groq — быстрый, бесплатный Claude — точный анализ DeepSeek
Рекомендация: для быстрых вопросов — Groq, для глубокого анализа и отчётов — Claude.

13. Статистика

Аналитические графики по всему портфелю проектов.

  • Распределение объектов по статусам и типам
  • Динамика завершения объектов по месяцам
  • Нагрузка на менеджеров (активные объекты)
  • Доля просроченных объектов в портфеле

14. Экспорт отчётов

Из большинства разделов доступна выгрузка данных в Excel одним кликом.

РазделЧто экспортируется
ПроектыПолный реестр с датами, статусами, менеджерами
ДедлайныТаблица этапов по всем активным объектам
ЗадачиСписок с исполнителями, приоритетами и статусами
ВПКРеестр отчётов по внутреннему контролю

Кнопка «Экспорт» или «Скачать Excel» находится в правом верхнем углу каждого раздела.

15. Для администратора

Только для администраторов: Гаврин Игорь и Месмер Денис.

Добавление пользователя

  1. Перейдите в меню → «Пользователи» (внизу боковой панели)
  2. Введите номер телефона нового сотрудника
  3. Укажите отображаемое имя и уровень прав
  4. Сохраните — сотрудник может войти сразу

Сброс пароля

Список пользователей → кнопка «Сбросить пароль». При следующем входе сотрудник задаст новый пароль.

Удаление доступа

Удалите телефон из белого списка — пользователь немедленно теряет доступ к системе.

Журнал действий

Страница /admin/audit — полная история всех действий: входы, создание и удаление объектов, смены статусов.

Важно: перед удалением пользователя убедитесь, что его объекты переназначены другому менеджеру.
