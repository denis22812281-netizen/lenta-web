1. Назначение системы

Лента.PM — внутренний инструмент строительного департамента сети ЛЕНТА. Обеспечивает полный контроль жизненного цикла объектов реконструкции и нового строительства от планирования до открытия.

Управление объектамиРеконструкция и Констракшн с историей изменений
Контроль дедлайновЭтапы проектов с цветовой индикацией состояния
Задачи и порученияВнутренний трекер с приоритетами и исполнителями
СМР-уведомленияАвтоматические письма подрядчикам с подтверждением
Карточки адаптацииПошаговый ввод данных по новым объектам
AI-ассистентОтветы на вопросы по данным системы в реальном времени

Глобальный поискМгновенный поиск по всем объектам системы
Команда: директор Комаров Алексей, руководитель Гаврин Игорь + 8 менеджеров проектов. Доступ предоставляется администратором по номеру телефона.

2. Вход в систему

Первый вход

  1. Откройте браузер и перейдите по адресу системы
  2. Введите номер телефона в формате +7XXXXXXXXXX
  3. Если номер зарегистрирован — введите пароль. При первом входе придумайте пароль — он сохранится и будет использоваться всегда
  4. Нажмите «Войти» — откроется главный дашборд
Важно: войти можно только с номера телефона, заранее добавленного администратором. Если номер не принимается — обратитесь к Месмеру Денису.

Безопасность входа

Белый списокТолько добавленные номера могут войти в систему
Защита от перебораНе более 5 попыток входа в минуту
Шифрование пароляПароль хранится в зашифрованном виде, сотрудники системы его не видят
СессияСессия активна 30 дней. При смене пароля — все сессии сбрасываются

Если забыли пароль

Обратитесь к администратору — Месмер Денис. Он сбросит аккаунт, при следующем входе вы зададите новый пароль самостоятельно.

Выход из системы

Кнопка «Выйти» в правом верхнем углу или внизу бокового меню. После выхода сессия немедленно завершается.

Совет: Добавьте сайт в закладки или на рабочий стол — на мобильном телефоне он работает как полноценное приложение (см. раздел 18).

4. Проекты — Реконструкция

Все объекты типа «Реконструкция» с фильтрацией, поиском и массовыми операциями.

Фильтрация и поиск

  • Фильтр по статусу, менеджеру, городу и году
  • Поиск по номеру ТК или названию объекта
  • Массовые действия — выбрать несколько объектов и изменить статус или менеджера разом

Статусы проекта

Планирование Активный Приостановлен Завершён

Создание объекта

  1. Нажмите «+ Добавить» в правом верхнем углу
  2. Укажите название, номер ТК, город, тип, статус
  3. Назначьте ответственного менеджера и плановые даты
  4. Нажмите «Сохранить»

Карточка объекта

Нажмите на объект для просмотра: основные данные, этапы с датами, файлы, комментарии, история изменений. В карточке доступна кнопка «Gantt» — визуализация графика этапов.

Комментарии: добавляйте заметки прямо в карточке проекта — они сохраняются с именем автора и временем.

5. Проекты — Констракшн

Объекты нового строительства. Интерфейс идентичен разделу «Реконструкция», за исключением специфичных полей.

Разница: в таблицах дедлайнов и исполнительном дашборде Реконструкции и Констракшн отображаются раздельно. Аналитика по открытиям ТК строится именно по объектам Констракшн.

Дата открытия

Ключевое поле для Констракшн — Дата открытия. Сравнивается с плановой датой и определяет попадание в категории «раньше срока», «вовремя», «с опозданием» в исполнительном дашборде и аналитике.

6. Задачи

Внутренний трекер команды для постановки и контроля поручений.

Создание задачи

  1. Раздел «Задачи» в боковом меню → «+ Новая задача»
  2. Укажите название, описание, приоритет и срок
  3. Назначьте исполнителя из списка менеджеров
  4. Нажмите «Сохранить»

Приоритеты и статусы

Низкий Средний Высокий
Открытая В работе На проверке Завершена
Совет: Завершая задачу, добавьте комментарий — это ведёт историю работы по объекту для будущих проверок.

7. Дедлайны

Сводная таблица по этапам всех активных проектов — одна страница показывает состояние по всему портфелю.

Как читать таблицу

  • Строки — объекты (ТК)
  • Столбцы — этапы проекта (Старт / Окончание)
  • Цвет ячейки — состояние этапа: просрочено / скоро / выполнено / в плане

Вкладки

Дедлайны разделены на вкладки: Реконструкции и Констракшн — переключайтесь между ними сверху страницы.

Импорт из Excel

  1. Подготовьте файл в установленном формате (столбцы: ТК, этапы, Старт, Окончание)
  2. Перейдите «Дедлайны» → «Импортировать»
  3. Выберите файл (макс. 10 МБ) и нажмите «Загрузить»
  4. Система автоматически обновит даты по всем объектам из файла
Формат Excel: уточните актуальный шаблон у Месмера Дениса — он адаптирован под внутреннюю структуру файлов ТК.

8. СМР — строительно-монтажные работы

Управление графиками подрядчиков и автоматическая отправка уведомлений о выполненных работах.

Отправка уведомления

  1. Откройте нужный объект из списка СМР
  2. Отметьте выполненные этапы работ
  3. Нажмите «Отправить уведомление» — подрядчик получит письмо со ссылкой
  4. Подрядчик переходит по ссылке и подтверждает выполнение работ
Подтверждение фиксируется в системе с датой и временем — автоматически для отчётности и контроля.

База контактов

Раздел «Контакты» → список подрядчиков с email-адресами. Добавляйте и обновляйте контакты перед отправкой уведомлений.

Еженедельный дайджест

Система автоматически формирует и отправляет еженедельный дайджест по статусу СМР-объектов руководителю.

9. ВПК — внутренний контроль

Мониторинг отчётов по объектам для внутреннего контроля качества строительства.

  • Реестр отчётов с датами, статусами и ответственными по объектам
  • Счётчик непрочитанных материалов — отображается в боковом меню
  • Предпросмотр отчётов без скачивания
  • Экспорт реестра в Excel одним кликом
Индикатор ВПК: цифра рядом с пунктом меню — количество новых непрочитанных отчётов. Обнуляется при просмотре.

Предосмотр ВПК

  1. Нажать «Новый предосмотр» → выбрать объект и тип (ВПК-1 / ВПК-2)
  2. Пройти по чеклисту критериев — отметить каждый пункт
  3. Прикрепить фотографии к нужным позициям
  4. Нажать «Отправить» — отчёт сохраняется и уходит на email

Галерея фото открытия

Раздел «Фото открытия» на странице объекта — отдельная галерея фотографий с церемонии открытия или с объекта.

  • Загрузить фото с телефона прямо в браузере (до 10 МБ каждое)
  • Пометить ключевые фото звёздочкой ★ — они выделяются в галерее
  • Кнопка «Скачать ZIP» — архив всех фото объекта одним файлом
  • Удалить отдельное фото кнопкой корзины

10. КСО — отчётность

Реестр КСО-документации — хранение и контроль отчётных документов по объектам.

  • Фильтрация по объекту, дате, статусу проверки
  • Прикрепление файлов к записям
  • Отметки о прохождении проверки с датой

11. Менеджеры

Справочник команды — контактные данные и текущая нагрузка каждого менеджера.

  • ФИО, телефон, email каждого сотрудника
  • Количество активных объектов у менеджера в реальном времени
  • Отметка руководителя (Гаврин Игорь) — выделена визуально

Менеджеры из этого справочника назначаются на объекты в карточках проектов. Нагрузка пересчитывается автоматически.

12. Карточки адаптации

НОВЫЙ РАЗДЕЛ

Модуль для подготовки комплектов адаптационных данных по новым объектам: оборудование кухни, холодильные витрины, мебель, вывески, кассы и прочее.

Пошаговый мастер

Карточка заполняется в 5 шагов — прогресс-бар вверху показывает текущий этап и количество заполненных полей на каждом шаге:

  1. Общая информация — номер ТК, город, тип реконструкции, менеджер, даты
  2. Кухня и оборудование ТЗ — холодильные витрины, высота помещения, состав оборудования
  3. Мебель и освещение — кассовые боксы, стойки, стеллажи, торговое оборудование
  4. Вывески и навигация — фасадные вывески, внутренняя навигация, указатели
  5. Итоги и комментарии — дополнительные пожелания, финальная проверка

Обучающий тур

При первом открытии формы автоматически запускается краткий тур — он объяснит назначение каждого шага. Повторно вызвать тур можно кнопкой в правом верхнем углу формы.

Сохранение и отправка

Черновик — сохранено, не отправлено Отправлено — финальная версия

Карточку можно сохранять как черновик и возвращаться к заполнению позже. Кнопка «Отправить» переводит её в финальный статус.

Скачать Excel

Нажмите «Скачать Excel» в карточке — система сформирует файл с заполненными данными в стандартном формате.

Совет: заполняйте обязательный минимум (Шаг 1) сразу при создании объекта, остальные шаги — по мере поступления данных.

15. Исполнительный дашборд  ТОП

Доступ: только Комаров Алексей (директор) и Гаврин Игорь (руководитель). Обозначен жёлтым бейджем «ТОП» в меню.

Аналитический дашборд для руководства — итоги портфеля Констракшн с акцентом на эффективность и ответственность менеджеров.

KPI-карточки

  • Открыто в году — Констракшн-объекты, открытые с начала текущего года
  • В работе сейчас — активные объекты Констракшн
  • Просрочено открытий — дата открытия позже плановой
  • Откроется в 90 дней — прогноз по плановым датам

Рейтинг менеджеров

Автоматически рассчитывается по формуле: открытия × 3 − просрочки × 5 − просроченные задачи × 2. Объективная оценка эффективности за год.

PDF для совещания

Кнопка «PDF» в правом верхнем углу — распечатайте дашборд прямо в браузере для совещания или отчёта директору.

18. Мобильная версия

НОВЫЙ РАЗДЕЛ

Система полностью адаптирована для работы со смартфона — ничего устанавливать не нужно, достаточно открыть браузер.

Нижняя навигация

На телефоне в нижней части экрана постоянно видна панель быстрого доступа:

Дашборд Дедлайны ВПК Менеджеры СМР

Боковое меню

Нажмите иконку в верхнем левом углу — откроется полное меню со всеми разделами. Закройте свайпом влево или кнопкой ×.

Добавить на рабочий стол

  1. Откройте систему в браузере Chrome или Safari
  2. Нажмите «Поделиться» (Safari) или «⋮» (Chrome) → «На главный экран»
  3. Подтвердите добавление — иконка появится как приложение
  4. Запускается без адресной строки, как нативное приложение
Уведомления: если браузер запросит разрешение на уведомления — разрешите, чтобы получать оповещения о срочных дедлайнах.

19. Чат

НОВЫЙ

Встроенный мессенджер команды — общий чат и личные сообщения без выхода из системы.

Отправка сообщений

  1. Раздел «Чат» в боковом меню
  2. Ввести текст в поле внизу → Enter или кнопка «Отправить»
  3. Прикрепить фото: иконка скрепки → выбрать изображение

Общий и личный чат

Общий чатВиден всем пользователям системы — для рабочих обсуждений
Личные сообщенияНажать имя коллеги в правой панели → открывается приватный диалог

Онлайн-статусы

Зелёная точка рядом с именем — коллега активен в системе прямо сейчас. Обновляется автоматически.

Уведомления: при новом сообщении приходит push-уведомление, если вкладка закрыта. Красный счётчик на иконке чата показывает число непрочитанных.

20. Push-уведомления

НОВЫЙ

Система присылает оповещения прямо на телефон — даже когда браузер закрыт.

Включить уведомления

  1. При первом входе браузер покажет запрос — нажать «Разрешить»
  2. Если пропустили: нажать иконку 🔔 в шапке → «Включить уведомления»

Какие уведомления приходят

СобытиеТекст уведомления

Дедлайн через 3 дня«ТК XXX — срок через 3 дня»

Выключить уведомления

Настройки браузера → Уведомления → найти адрес системы → «Заблокировать».

iPhone: push-уведомления работают только если сайт добавлен на рабочий стол как PWA (Safari → Поделиться → «На экран Домой»). В обычном браузере Safari на iOS они не работают.

21. AI-конвертер (Фото → Excel)

НОВЫЙ
Доступ: только администраторы. Раздел «Инструменты» в боковом меню.

Загружаете фото акта, таблицы или рукописных данных — AI распознаёт содержимое и создаёт готовый файл Excel.

Режимы конвертации

РежимЧто делает
ФотоОдно фото (акт, накладная, таблица) → Excel
ТекстВставить текст или продиктовать голосом → Excel
Несколько фотоДо 5 фото → один Excel с несколькими листами
СравнениеДва фото → Excel с выделенными различиями (жёлтым)
ОбъединениеДо 5 фото → один общий лист

Типы результата

📋 Таблица — стандартная с заголовками 📊 Диаграмма — данные + бар-чарт 🥧 Круговая — данные + пирог 📅 Ганта — временная шкала задач

Пошагово (режим Фото)

  1. Выбрать режим «Фото»
  2. Перетащить фото в зону загрузки или нажать на неё
  3. Выбрать тип результата (Таблица / Диаграмма / …)
  4. Нажать «Конвертировать» — ждать до 45 секунд
  5. Excel скачается автоматически

Голосовой ввод (режим Текст)

  1. Выбрать режим «Текст»
  2. Нажать иконку микрофона 🎤 → разрешить доступ к микрофону
  3. Говорить — текст появляется в поле в реальном времени
  4. Нажать микрофон снова чтобы остановить → «Конвертировать»
Голос работает только в Chrome/Edge. Если AI не отвечает больше 90 секунд — система покажет ошибку. Попробуйте ещё раз.

История конвертаций

Раздел «Последние конвертации» внизу страницы — хранится 10 последних файлов в течение 24 часов.

⬇ — скачать повторно 🗑 — удалить из истории

22. Для администратора

Доступ к этому разделу: Гаврин Игорь и Месмер Денис.

Добавление пользователя

  1. Меню → «Пользователи» (внизу боковой панели) → «Добавить»
  2. Введите номер телефона нового сотрудника
  3. Укажите отображаемое имя и уровень прав (admin / обычный)
  4. Сохраните — сотрудник может войти сразу

Сброс пароля

Список пользователей → кнопка «Сбросить пароль». При следующем входе сотрудник задаст новый пароль самостоятельно.

Удаление доступа

Удалите телефон из белого списка — пользователь немедленно теряет доступ к системе. Все его объекты остаются в базе.

Важно: перед удалением пользователя убедитесь, что его объекты переназначены другому менеджеру.

Журнал действий

Страница /admin/audit — полная история всех событий: входы, создание и удаление объектов, смены статусов с именем пользователя и IP-адресом.

IP-ограничение для администраторов

Страницы администратора (/admin/…) могут быть закрыты по IP-адресу — доступ только с разрешённых сетей. Настраивается через переменную окружения ADMIN_IP_WHITELIST. Обратитесь к разработчику при необходимости смены адресов.

Шаблон карточки адаптации

Страница /admin/adaptation-template — настройка состава полей и опций для карточек адаптации. Изменения применяются ко всем новым карточкам.
