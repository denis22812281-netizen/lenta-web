1. Назначение системы

Система управления проектами Лента ПМ — внутренний инструмент команды строительного департамента ЛЕНТА. Предназначена для:

  • Отслеживания статусов объектов реконструкции и констракшн
  • Контроля дедлайнов по этапам проектов
  • Управления задачами внутри команды
  • Хранения и отправки SMR-уведомлений подрядчикам
  • Формирования отчётов и экспорта в Excel
  • AI-аналитики текущего состояния портфеля
Команда: руководитель Гаврин Игорь + 8 менеджеров проектов. Доступ — по номеру телефона, утверждённому администратором.
🔑

2. Вход в систему

Первый вход

  1. Откройте браузер и перейдите по ссылке системы
  2. Введите свой номер телефона в формате +7XXXXXXXXXX
  3. Если номер есть в системе — введите пароль (при первом входе создайте его)
  4. Нажмите «Войти» — вы попадёте на главный дашборд
Совет: Сохраните пароль в менеджере паролей браузера — кнопка появится при вводе.

Если забыли пароль

Обратитесь к администратору системы (Месмер Денис). Он сбросит пароль, и вы сможете установить новый при следующем входе.

QR-код для быстрого входа

На странице входа доступна страница /qr — QR-код со ссылкой на систему для быстрого открытия с мобильного.

🏠

3. Дашборд (главная страница)

Главная страница показывает текущее состояние всего портфеля проектов:

  • Карточки статистики — количество активных, завершённых, просроченных объектов
  • Срочные объекты — проекты с дедлайном менее 7 дней
  • Просроченные проекты — объекты, у которых истёк срок
  • Открытые задачи — задачи, требующие внимания
  • Онлайн — кто из команды сейчас в системе
Цветовая индикация: красный — просрочено, жёлтый — срок менее 7 дней, зелёный — всё в порядке.
🔨

4. Проекты — Реконструкция

Раздел «Реконструкции» содержит все объекты типа «Реконструкция».

Список объектов

  • Фильтрация по статусу, менеджеру, городу
  • Поиск по номеру ТК или названию
  • Массовые действия — выбрать несколько объектов и изменить статус или менеджера

Карточка объекта

Нажмите на объект — откроется детальная карточка:

  • Основные данные: ТК, город, тип, статус, менеджер
  • Даты: начало работ, плановое окончание, дата открытия
  • Этапы проекта с датами и отметками выполнения
  • Файлы и фотографии
  • История изменений

Статусы проекта

Планирование Активный Приостановлен Завершён

Создание нового объекта

  1. Нажмите кнопку «+ Добавить» в правом верхнем углу
  2. Заполните обязательные поля: название, номер ТК, город, тип, статус
  3. Назначьте менеджера и укажите даты
  4. Нажмите «Сохранить»
🏗

5. Проекты — Констракшн

Раздел «Констракшн» — аналогичен Реконструкциям, но для новых объектов строительства. Интерфейс и логика работы идентичны.

Разница: в дедлайн-таблицах и отчётах Реконструкции и Констракшн отображаются отдельными вкладками.
✅

6. Задачи

Раздел «Задачи» — внутренний трекер для команды. Задачи не привязаны к конкретному проекту и используются для оперативного взаимодействия.

Создание задачи

  1. Перейдите в раздел «Задачи» в боковом меню
  2. Нажмите «+ Новая задача»
  3. Укажите название, описание, приоритет, исполнителя и срок
  4. Сохраните

Приоритеты

Низкий Средний Высокий

Изменение статуса задачи

Откройте задачу → кнопка «Изменить статус» → выберите новый статус → при необходимости оставьте комментарий.

Совет: При завершении задачи добавьте комментарий — это помогает вести историю работы.
📅

7. Дедлайны

Раздел «Дедлайны» — сводная таблица по этапам всех активных проектов. Показывает какие стадии просрочены, какие завершены, какие в работе.

Что показывает таблица

  • По строкам — объекты (ТК)
  • По столбцам — этапы проекта с датами начала и окончания
  • Цветовая подсветка: красный — просрочено, жёлтый — скоро, зелёный — выполнено

Импорт из Excel

  1. Подготовьте файл Excel в установленном формате
  2. Перейдите в «Дедлайны» → «Импортировать»
  3. Выберите файл и нажмите «Загрузить»
  4. Система обновит даты по всем объектам автоматически
Лимит файла: максимальный размер импортируемого файла — 10 МБ.

Экспорт

Кнопка «Экспорт Excel» → скачивается файл с текущим состоянием всех дедлайнов.

🔧

8. SMR — строительно-монтажные работы

Модуль SMR используется для управления графиками подрядчиков и отправки уведомлений о выполненных работах.

Список SMR-графиков

Каждый объект может иметь один SMR-график с задачами по этапам работ.

Отправка уведомления подрядчику

  1. Откройте нужный SMR-объект
  2. Отметьте выполненные задачи
  3. Нажмите «Отправить уведомление» — подрядчик получит email с деталями
  4. Подрядчик переходит по ссылке и подтверждает выполнение работ

База контактов SMR

Раздел «Контакты» → список всех подрядчиков с email для рассылок. Можно добавлять и удалять контакты.

Подтверждение: после отправки уведомления система фиксирует факт и дату подтверждения — для отчётности.
📊

9. ВПК — внутренний контроль

Раздел ВПК — мониторинг отчётов и уведомлений по объектам для внутреннего контроля.

  • Список отчётов по объектам с датами
  • Индикатор непрочитанных материалов
  • Экспорт реестра в Excel
Индикатор: в боковом меню рядом с «ВПК» отображается счётчик непрочитанных отчётов.
📁

10. КСО — отчётность

Раздел КСО содержит реестр отчётных документов по объектам. Используется для хранения и контроля КСО-документации.

  • Фильтрация по объекту, дате, статусу
  • Прикрепление файлов к записям
  • Отметки о проверке
👥

11. Менеджеры

Раздел «Менеджеры» — справочник команды с контактными данными и нагрузкой.

  • ФИО, телефон, email каждого менеджера
  • Количество активных объектов на менеджере
  • Кто является руководителем (Гаврин Игорь)

Менеджеры назначаются на объекты из этого справочника.

🤖

12. AI-ассистент

Встроенный AI-ассистент — чат-бот, который видит все данные системы в реальном времени и отвечает на вопросы о проектах.

Что умеет

  • «Сколько у нас просроченных объектов?»
  • «Какие объекты закроются в этом месяце?»
  • «У кого из менеджеров больше всего активных проектов?»
  • «Составь сводку по объектам за неделю»
  • Анализировать загруженные Excel-файлы на ошибки

13. Статистика

Раздел «Статистика» — графики и аналитика по портфелю проектов.

  • Распределение объектов по статусам и типам
  • Динамика завершения по месяцам
  • Нагрузка на менеджеров
  • Доля просроченных объектов
Доступ: статистика доступна всем авторизованным пользователям.

15. Для администратора

Только для администраторов — Гаврин Игорь и Месмер Денис.

Управление пользователями (/admin/users)

  1. Перейдите в меню → «Администрирование»
  2. Добавьте номер телефона нового сотрудника
  3. Укажите отображаемое имя и права (обычный / администратор)
  4. Сотрудник сможет войти, указав этот телефон

Сброс пароля

В списке пользователей нажмите «Сбросить пароль» напротив нужного сотрудника. При следующем входе пользователь создаст новый пароль.

Удаление пользователя

Удалите телефон из белого списка — пользователь сразу потеряет доступ к системе.

Журнал действий (/admin/audit)

Доступен только главному администратору. Показывает все действия пользователей: входы, создание и удаление проектов, изменения статусов.

Важно: перед удалением пользователя убедитесь, что его объекты переназначены другому менеджеру.
