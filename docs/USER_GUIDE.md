## 1. Первый вход и установка приложения

### Вход

1. Открыть браузер и перейти по ссылке на систему
2. Ввести номер телефона в формате **+7XXXXXXXXXX**
3. Если номер добавлен администратором → появится поле пароля
4. **Первый вход:** ввести любой пароль — он станет вашим постоянным паролем
5. При последующих входах использовать этот же пароль

> ⚠️ Если номер телефона не принимается — обратитесь к администратору (Месмер Денис). Вход возможен только с заранее добавленных номеров.

### Выход

Кнопка **«Выйти»** в правом верхнем углу или в боковом меню.

## 2. Главная страница (дашборд)

Показывает общий срез по всем проектам:

- **Карточки статистики** — всего проектов, просроченных, по статусам
- **Ближайшие дедлайны** — проекты с истекающим сроком (выделяются красным)
- **Последние активные проекты** — быстрый переход

### Переключение темы

Иконка луны / солнца в шапке — переключает светлую и тёмную тему.

## 3. Проекты

Раздел `/projects` — два типа: **Реконструкция** и **Констракшн**.

### Создание проекта

1. Нажать **«+ Новый проект»**
2. Заполнить: номер ТК, название, тип, статус, менеджер, даты
3. Нажать **«Сохранить»**

### Редактирование

1. Открыть проект → кнопка **«Редактировать»**
2. Изменить поля → **«Сохранить»**
3. Все изменения записываются в **Историю** (вкладка на странице проекта)

### Этапы и диаграмма Ганта

- Вкладка **«Ганта»** на странице проекта — интерактивная диаграмма
- Перетащить полоску этапа чтобы изменить дату — изменения сохраняются автоматически
- Добавить этап: кнопка **«+ Этап»** → ввести название и даты

### Вложения

- Вкладка **«Вложения»** → кнопка **«Прикрепить файл»**
- Поддерживаются: PDF, DOCX, XLSX, изображения (до 20 МБ)
- Каждый файл можно скачать или удалить

### Комментарии

- Вкладка **«Комментарии»** → ввести текст → Enter или кнопка **«Отправить»**
- Комментарии видны всем пользователям с доступом к проекту

### Экспорт в Excel

Кнопка **«Экспорт»** на странице списка проектов — скачивает XLSX со всеми данными.

### Импорт из Excel

- Кнопка **«Импорт»** → выбрать файл XLSX
- Формат: столбцы должны совпадать с шаблоном (скачать шаблон можно там же)
- После импорта — проверить данные, ошибки выводятся списком

## 4. Задачи

Раздел `/tasks` — задачи для всей команды.

### Создание задачи

1. Нажать **«+ Задача»**
2. Заполнить: название, описание, исполнитель, приоритет (низкий / средний / высокий), дедлайн
3. Нажать **«Создать»**

### Приоритеты

| Цвет | Приоритет |
|------|----------|
| Серый | Низкий |
| Жёлтый | Средний |
| Красный | Высокий |

### Смена статуса

Нажать кнопку статуса на карточке задачи:
- **Новая** → **В работе** → **Выполнена** → **Отменена**

### Дедлайны задач

Просроченные задачи выделяются красным. Все дедлайны — в разделе [Дедлайны](#8-дедлайны).

## 5. ВПК-отчёты

Раздел `/vpk` — отчёты о посещении объектов (ВПК-1 и ВПК-2).

### Создание отчёта (Предосмотр ВПК)

1. Нажать **«Новый предосмотр»**
2. Выбрать объект и тип (ВПК-1 / ВПК-2)
3. Пройти по чеклисту — отметить каждый критерий (выполнено / не выполнено / замечание)
4. Прикрепить фотографии к позициям где нужно
5. Нажать **«Отправить»** — отчёт уходит на email и сохраняется в системе

### Просмотр отчётов

- Список всех отчётов с фильтрацией по объекту, дате, статусу
- Непрочитанные помечаются жёлтой точкой
- Нажать **«Отметить прочитанным»** чтобы убрать метку

### Галерея открытий

- Раздел **«Фото открытия»** на странице объекта
- Загрузить фото с телефона прямо в браузере
- Каждому фото можно поставить метку «Ключевое» (звёздочка)
- Кнопка **«Скачать ZIP»** — архив всех фото объекта

### Экспорт

Кнопка **«Экспорт Excel»** на странице отчётов — XLSX со сводкой.

## 6. График СМР

Раздел `/smr` — управление графиком строительно-монтажных работ.

### Просмотр графика

- Список объектов с временной шкалой задач
- Цветовая кодировка: зелёный — выполнено, жёлтый — в работе, красный — просрочено

### Смена статуса задачи СМР

1. Открыть задачу → кнопка статуса
2. Выбрать новый статус → подтвердить

### Email-подтверждения

- Кнопка **«Отправить подтверждение»** — уходит письмо на email менеджера объекта
- Менеджер подтверждает выполнение по ссылке в письме
- Статус задачи автоматически меняется на «Подтверждено»

### Отчёт по объекту

- Кнопка **«Отправить отчёт»** → ввести email получателя → **«Отправить»**
- Получатель получает письмо с PDF-сводкой по объекту

### Поиск контактов

Поле поиска в разделе СМР — ищет по имени и телефону менеджеров в белом списке.

## 8. Дедлайны

Раздел `/deadlines` — единый экран всех сроков.

- **Реконструкция** и **Констракшн** — отдельные вкладки
- Просроченные выделяются красным, истекающие сегодня — жёлтым
- Push-уведомление приходит за **3 дня** до дедлайна

### Экспорт

Кнопка **«Экспорт»** — XLSX со всеми дедлайнами и статусами.

## 10. Чат

Раздел `/chat` — мессенджер команды.

### Отправка сообщения

- Ввести текст в поле внизу → Enter или кнопка **«Отправить»**
- Прикрепить фото: иконка скрепки → выбрать изображение

### Каналы

- **Общий чат** — виден всем пользователям
- **Личные сообщения** — нажать имя пользователя в списке справа

### Онлайн-статус

Зелёная точка рядом с именем — пользователь онлайн (активен в последние 5 минут).

### Уведомления

Push-уведомление приходит при новом сообщении если вкладка закрыта.  
Красный счётчик на иконке чата в меню показывает число непрочитанных.

## 11. ИИ-конвертер (Фото → Excel)

Раздел `/tools/photo-to-excel` — конвертация данных в таблицы Excel с помощью AI.

> Доступен только администраторам.

### Режимы работы

| Режим | Что делает |
|-------|-----------|
| **Фото** | Одно фото (акт, таблица, накладная) → Excel |
| **Текст** | Вставить текст или продиктовать голосом → Excel |
| **Несколько фото** | До 5 фото → один Excel с несколькими листами |
| **Сравнение** | Два фото → Excel с выделенными различиями |
| **Объединение** | До 5 фото → один общий лист |

### Типы результата

| Тип | Описание |
|-----|---------|
| **Таблица** | Стандартная таблица с заголовками |
| **Диаграмма** | Данные + встроенная диаграмма |
| **Круговая** | Данные + круговая диаграмма |
| **Ганта** | Временная шкала задач |

### Пошагово (режим Фото)

1. Выбрать режим **«Фото»**
2. Нажать на зону загрузки или перетащить фото
3. Выбрать тип результата (таблица / диаграмма / …)
4. Нажать **«Конвертировать»**
5. Дождаться (до 45 секунд) → Excel скачается автоматически

### Голосовой ввод (режим Текст)

1. Выбрать режим **«Текст»**
2. Нажать иконку микрофона 🎤
3. Говорить — текст появляется в поле в реальном времени
4. Нажать микрофон снова чтобы остановить
5. Нажать **«Конвертировать»**

> ⚠️ Голосовой ввод работает только в Chrome/Edge. Требует разрешения микрофона.

### История конвертаций

Раздел **«Последние конвертации»** внизу страницы:
- Кнопка **⬇** — скачать повторно (хранится 24 часа, max 10 последних)
- Кнопка **🗑** — удалить из истории

### Шаблоны

- Нажать **«Сохранить шаблон»** после успешной конвертации
- Шаблон запоминает режим и тип результата
- В следующий раз — выбрать шаблон и сразу загружать файл

## 14. Push-уведомления

Push-уведомления позволяют получать оповещения даже когда браузер закрыт.

### Включить уведомления

1. При первом входе браузер спросит разрешение — нажать **«Разрешить»**
2. Если пропустили — нажать иконку 🔔 в шапке → **«Включить уведомления»**

### Что приходит

| Событие | Уведомление |
|---------|------------|
| Новое сообщение в чате | «Имя: текст сообщения» |
| Дедлайн через 3 дня | «ТК XXX — срок через 3 дня» |
| Просроченный дедлайн | «ТК XXX — просрочен на N дней» |
| Новый ВПК-отчёт | «Новый отчёт по объекту XXX» |

### Выключить уведомления

Настройки браузера → Уведомления → найти адрес системы → **«Заблокировать»**.

## 15. Мобильное приложение (PWA)

Лента.PM можно установить на телефон как обычное приложение.

### Установка на Android (Chrome)

1. Открыть систему в Chrome
2. Нажать меню **⋮** (три точки) → **«Добавить на главный экран»**
3. Подтвердить установку
4. Иконка появится на рабочем столе

### Установка на iPhone (Safari)

1. Открыть систему в Safari
2. Нажать кнопку **«Поделиться»** (квадрат со стрелкой вверх)
3. Выбрать **«На экран «Домой»»**
4. Нажать **«Добавить»**

### Особенности мобильной версии

- Нижняя навигация (вместо бокового меню)
- Оптимизированы таблицы и формы под маленький экран
- Push-уведомления работают в фоне

## 16. Администрирование

> Этот раздел — только для пользователей с правами администратора.

### Пользователи `/admin/users`

- **Список пользователей** — все зарегистрированные и добавленные номера
- **Добавить** → ввести телефон, имя, поставить галочку «Администратор» если нужно
- **Удалить** → кнопка «Удалить» на строке пользователя (подтверждение обязательно)
- Пользователь без записи в белом списке не сможет войти

### Критерии ВПК `/admin/vpk-criteria`

- Список критериев чеклиста ВПК-1 и ВПК-2
- Перетащить строку чтобы изменить порядок
- **Добавить критерий** → кнопка «+ Критерий» → ввести текст → Enter
- **Удалить** → кнопка корзины на строке критерия

### Журнал аудита `/admin/audit`

- Лог всех действий пользователей: входы, изменения, удаления
- Фильтрация по дате, пользователю, типу действия
- Нельзя редактировать (только просмотр)

### Резервная копия `/admin/backup`

- Нажать **«Создать бэкап»** — создаст дамп PostgreSQL
- Файл `.sql` скачивается в браузер
- Рекомендуется делать бэкап перед крупными обновлениями

## Частые вопросы

**Забыл пароль — что делать?**  
Обратитесь к администратору (Месмер Денис). Администратор может сбросить аккаунт через `/admin/users` → удалить пользователя → при следующем входе установить новый пароль.

**Не приходят push-уведомления на iPhone**  
Push на iOS требует Safari 16.4+ и установки как PWA (Добавить на экран «Домой»). В обычном браузере Safari на iOS push-уведомления не работают.

**AI-конвертер завис (крутится больше 90 секунд)**  
После 90 секунд система показывает ошибку «AI не ответил». Попробуйте ещё раз — обычно это временная перегрузка AI-сервиса.

**Фото не загружается**  
- Максимальный размер: 20 МБ
- Поддерживаемые форматы: JPG, PNG, WEBP, PDF
- Проверьте интернет-соединение

**Вкладка показывает ошибку 500**  
Нажать Ctrl+F5 (жёсткое обновление). Если не помогает — написать Месмеру Денису на denis.mesmer@lenta.com.
