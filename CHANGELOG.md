# История версий — Лента.PM

## v1.0.0 (2026)

### Первый релиз

**Управление проектами**
- Импорт проектов из Excel (Реконструкции, Констракшн, КСО)
- Автоматическая синхронизация с Excel-файлами
- Отслеживание этапов: СИД, Зонирование, ВПК, Открытие
- Статусы проектов с автоматическим расчётом по датам
- Экспорт в Excel с цветовой разметкой

**Задачи**
- Постановка задач с дедлайнами и приоритетами
- Уведомления исполнителю и постановщику
- AJAX-фильтрация без перезагрузки страницы
- Комментарий при завершении задачи

**ВПК-отчёты**
- Чеклисты ВПК-1 (17 критериев) и ВПК-2 (13 критериев)
- Загрузка фотографий к каждому критерию
- Экспорт отчётов в Excel
- Уведомления руководителю о новых отчётах

**Аналитика**
- Статистика открытий: раньше срока / вовремя / с опозданием
- Дашборд с ближайшими дедлайнами
- Отчёт по просроченным задачам и проектам

**Коммуникации**
- Внутренний чат (личные сообщения + общий)
- Онлайн-статусы пользователей в реальном времени
- Отправка фотографий в чате

**ИИ-ассистент**
- Поддержка 3 провайдеров: Claude, Groq, DeepSeek
- Контекст с актуальными данными о проектах
- Проверка Excel-файлов на ошибки формул

**Безопасность**
- Вход по номеру телефона + пароль
- Face ID / Touch ID / Windows Hello (WebAuthn FIDO2)
- CSRF-защита всех форм
- Rate limiting: 5 попыток/мин на ввод пароля
- Пароли: PBKDF2-SHA256 с солью (260 000 итераций)

**Инфраструктура**
- PWA: можно установить на экран телефона
- Push-уведомления в браузере
- Поддержка PostgreSQL и SQLite
- Деплой на Railway.app
- Автосинхронизация Excel-файлов по расписанию
